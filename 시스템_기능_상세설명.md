# 🚀 뉴스 큐레이션 및 AI 보고서 시스템 기능 상세 설명

본 문서는 시스템을 구성하는 주요 모듈별 세부 기능과 작동 원리를 상세히 설명합니다.

---

## 1. 뉴스 데이터 수집 모듈 (`01_fetch_news.py`)
- **기능**: 외부 뉴스 API(NewsAPI.org)를 통해 실시간 기사 데이터를 확보합니다.
- **상세**:
    - `config.py`에 설정된 `NEWS_QUERY_KEYWORD`를 기반으로 한국어 뉴스를 검색합니다.
    - 검색 범위: 경제, 반도체, 세계 경제, 전쟁 등 핵심 키워드 조합.
    - 결과물: 수집된 원본 데이터는 `primary_data/raw_news_YYYYMMDD.json`에 저장됩니다.

## 2. 거시경제 분석 리포트 엔진 (`05_generate_macro_report.py`)
- **기능**: 당일 수집된 전체 뉴스를 조망하여 거시경제 관점의 분석 보고서를 작성합니다.
- **핵심 기술 (Google Search Grounding)**:
    - 단순히 제공된 뉴스뿐만 아니라, Google 검색 기능을 실시간으로 활용하여 최신 지표와 배경 정보를 보강합니다.
- **분석 항목**:
    - 글로벌 주요 국가(미, 중, 유, 한)의 경제 상황 요약.
    - 금융시장(증시, 금리, 환율, 원자재) 흐름 분석.
    - 오늘 시장을 움직인 핵심 인과관계 도출 및 3가지 투자 시나리오(A/B/C) 제시.
- **키워드 추출**: 생성된 보고서에서 당일 가장 중요한 핵심 키워드 10개를 AI가 직접 추출하여 뉴스 필터링 단계로 전달합니다.

## 3. 지능형 뉴스 필터링 및 요약 (`02_filter_summarize.py`)
- **기능**: 수백 개의 뉴스 중 투자에 실질적인 영향을 주는 핵심 기사만 선별합니다.
- **작동 원리**:
    - **보고서 연동 필터링**: 위 단계에서 추출된 '오늘의 핵심 키워드'와 연관된 기사를 우선순위로 둡니다.
    - **AI 평가**: Gemini 모델이 각 기사를 읽고 `relevant=true/false`를 판단합니다.
    - **건조한 요약**: 학술적이고 객관적인 톤으로 3~4줄 요약을 수행하며, 관련 섹터를 자동 분류합니다.

## 4. 노션(Notion) 통합 관리 (`03_update_notion.py`, `06_upload_report_to_notion.py`)
- **기능**: 분석된 데이터를 사용자가 보기 편한 노션 워크스페이스로 전송합니다.
- **상세**:
    - **개별 뉴스 DB**: 갤러리 뷰에 최적화된 형태로 제목, 섹터, 요약, URL을 업로드합니다.
    - **브리핑 페이지**: 마크다운 형식의 전문 보고서를 페이지 본문에 생성합니다.
    - **자동 태깅**: AI가 분석한 보고서 키워드를 뉴스 기사에 자동으로 태깅하여 검색 편의성을 높입니다.

## 5. 자동 유지보수 및 클라우드 실행
- **데이터 정리 (`04_cleanup_notion.py`)**: 노션 워크스페이스의 쾌적함을 위해 10일이 지난 기사는 자동으로 아카이브(삭제) 처리합니다.
- **깃허브 액션 스케줄링**: 모든 프로세스는 내 PC가 꺼져 있어도 깃허브 서버에서 매일 오전 8시(KST)에 정해진 순서대로 실행됩니다.
